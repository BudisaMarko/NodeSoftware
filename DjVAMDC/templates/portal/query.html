How to use:

  • Select the parameters that you want to restrict and fill in lower and/or upper boundaries. Use the same value for upper and lower to achieve "equal".
  • Selecting a parameter and not entering any bounds means that there must be a value present for this parameter.
  • The values for upper and lower bounds are to be enteres as decimals with at most 8 significant digits. Exponents are allowed, e.g. 4.567E-8
  • In the second part of the form, you select what kind of data you want that matches the criteria. For the time being, all information about the selection is returned.
