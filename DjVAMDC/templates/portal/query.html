How to use:

  • Select the parameters that you want to restrict and fill in lower and/or upper boundaries. Use the same value for upper and lower to achieve "equal".
  • Selecting a parameter and not entering any bounds means that there must be a value present for this parameter.
  • The values for upper and lower bounds are to be enteres as decimals with at most 6 significant digits plus exponent, e.g. 4.567E-8
  • The check box that says "Use AND?" is marked by default which means that an AND operator will be used to connect the current condition to the next one. If you want to use an OR, untick the marker.
